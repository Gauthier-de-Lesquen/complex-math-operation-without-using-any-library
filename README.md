# complex-math-operation-without-using-any-library

this file includes the functions:
* ```sqrt(n)``` to calculate the square root
* ```nthrt(n, n)``` to calculate the nth root
* ```factorial(n)``` to calculate the factorial
* ```sum((1, "n"), 4, "2n")``` or ```sum([1, "n"], 4, "2n")``` in JS with (1, "n"): the start of the sum and the letter, 4: the end, and "2n" the expression.
* ```product((1, "n"), 4, "2n")``` or ```product([1, "n"], 4, "2n")``` in JS
* ```exp(n)``` exponential with Taylor Series
* ```ln(n)``` natural logarythm
* ```sin(n)```, ```cos(n)```, ```tan(n)```
* ```gcd(a, b)```, ```lcm(a, b)``` greatest common divisior and least common multiple
